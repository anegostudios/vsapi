Collection of unique ordered elements sorted in ascending order.
Thread-safe.

Pros:

  • no memory overhead
  • fast to compare (O(1))
  • fast to compare via set operations
  • fast to iterate
  • fast to check containment
  • cheap to copy and pass around

    Cons:
  • slow to construct
  • slow to merge via set operations
